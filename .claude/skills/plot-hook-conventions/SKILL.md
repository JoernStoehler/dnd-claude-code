# Hook Conventions

Format model: Sentient Cargo (preserved in git history as `campaigns/fun-heroes/sessions/hook-sentient-cargo.md`).

## Required sections

### 1. Header block
```
# Hook Title
**Status:** [Hook developed, not yet run | Run on DATE | Concept only]
**Estimated Runtime:** X-Y hours
```

### 2. The Situation
- One paragraph: what's happening
- One sentence: the "obvious secret" (what players will likely deduce)
- One sentence: what happens if players do nothing

### 3. Characters
For each significant NPC:
- Name, appearance, want, fear, quirk (inline or structured)
- Voice guide: how to perform them (vocal quality + vocabulary)
- Sample lines: 2-3 distinct dialogue examples

For ensemble casts (3+ NPCs with interrelationships): add a relationship matrix table.

### 4. Structure (location-based or scene-based)
- For location-based hooks: structure as location progression. Each location gets: scene-setting paragraph, potential encounters, NPC reactions.
- For scene-based hooks: structure as sequential beats with branch points.
- In both cases: consistent subsection format across all nodes.

### 5. Complications
- Named, modular chunks (usable at any point in the adventure unless sequence-dependent)
- Each complication: title, one-sentence description, NPC reactions
- Header: "Pick 2-3 to populate the session"

### 6. Outcome paths
- Label distinct outcomes (A/B/C/D) rather than describing as prose
- Each outcome: what changes, what consequences ripple forward

### 7. Post-session tracking
```
## Post-Session

### What Happened
[Fill after running]

### Now Nailed Down
[Facts established — these become canon]

### Loose Threads
[What might come back]

### What Worked / What Didn't
[GM notes for future improvement]
```
Add NPC/item fate tables if disposition matters for canon.

## Skimmability principles

- **Bold key terms** (Want, Fear, Useful, Annoying) so eyes jump to relevant info
- **Sample dialogue in quotes** — fastest reference during play
- **Tables over prose** for relationships, fates, complications
- **Consistent subsection format** across repeated elements (locations, NPCs, complications)
- **Three-tier readability:** immediately useful (first 30s) → referenceable (look up during play) → discoverable (read before session, referenced during)

## Discoverable depth

For hooks longer than 1 hour: include 1-2 subplot hooks per major NPC that emerge only with engagement. These are optional — if players ignore them, the surface adventure still works.

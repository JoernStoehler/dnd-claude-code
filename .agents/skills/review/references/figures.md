# Figures, Cards, And PDFs Review

Use for image generation, portrait cards, handout PDFs, and HTML-to-PDF output.

Check:

- The producing script and source manifest are named.
- The generated artifact exists at the path referenced by the task or README.
- Page size, margins, card dimensions, rows/columns, and image scaling match the
  intended print use.
- Text remains readable in the rendered PDF or HTML.
- Images are not stretched unless the script explicitly documents that choice.
- Secret/GM-only materials are labeled or stored so they are not confused with
  player-facing handouts.
- The review does not spend FAL/API credits unless the task approved generation.

When possible, inspect the rendered artifact directly. If visual inspection is
not possible, say which checks were limited to source code.
